shimmer

PURPOSE

Calculates the shimmer of the signal. Calculate its absolute value

SYNOPSIS

DESCRIPTION 

 Calculates the shimmer of the signal. Calculate its absolute value
 in dB (rShdB), and its relative value in % (rShim)

 Input parameters:

 Output parameters:
