CalculateRegularity

PURPOSE

Calculates a series of regularity features from the inputted attractor

SYNOPSIS

function [AE, SE, mSE, GE, FE] = CalculateRegularity( mAttractorp1, mAttractor,rParam, sTypeNorm, bFast )

DESCRIPTION 

 Calculates a series of regularity features from the inputted attractor
 mAttractorp1 of dimension (m+1), using the tolerance value rParam.
